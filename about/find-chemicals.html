Direct Search

Simply search function is available to search all Ambeed products by any keywords such as product name, product #, MDL No., CAS # in one search window.

After typing, the search window will automatically complete your keywords and display some options, including exact search products, a fuzzy search for similar products or categories.

Please see instructions below on how to use the Search Function.

ItemTips on How to Enter Keywords
CAS NumberHyphens can be abbreviated.
Enter full numbers. If a partial number is entered, the results will not be correct.
MDL Number"MFCD" can be abbreviated.
Enter full numbers including "0". If a partial number is entered, the results will not be correct.
CAT NumberEnter one letter followed numbers (e.g.A184890).
If a partial number is entered, the results will not be correct.

Structure Search

Instructions for using the structure search and drawing tool:

1. Hover over the icons to learn the function of the chemical structure drawing tools.

2. Use the canvas to construct, import, or edit your molecules. Click on an item in the pop-up menu to set atom and bond properties, attach data to atoms and bonds, or set query properties on atoms.

3. The tool contains four tool bars, placed around the drawing canvas:

  • ●A general toolbar (top) includes import, export, undo, cut, paste, view, and settings.
  • ●The drawing toolbar (left) can be used to clear the canvas or delete parts of an object. Draw by dragging from the toolbar and clicking on the canvas using any of the various bond types, symbols, or charges.
  • ●The atom toolbar (right) allows you to select an atom by clicking on one of the frequently-used atoms or select an atom from periodic table icon at the top of the toolbar.
